# multiplication Table Generator

A command-line tool used to generate a Multiplication Table and outputs it as a text file.

## Usage
multiplication row size column size\
Example\
multiplication 4 3\
will generate

1 X 1 = 1\
1 X 2 = 2\
1 X 3 = 3\
\
2 X 1 = 2\
2 X 2 = 4\
2 X 3 = 6\
\
3 X 1 = 3\
3 X 2 = 6\
3 X 3 = 9\
\
4 X 1 = 4\
4 X 2 = 8\
4 X 3 = 12
